# GINGER RUBY 4 FOM
Dieser Dockercontainer ist für die FOM Vorlesung Betriebssysteme

Docker starten:
```
docker-compose up -d
```
Verbinden mit dem Docker Container:
```
docker exec -it ginger_ruby bash
```
work directory:
```
cd /var/ruby/code/ 
```
Ruby App starten:
```
ruby app.rb
```
